# CSV Analyzer

Applicazione Python per analizzare file CSV, filtrare dati, calcolare statistiche e generare grafici tramite Streamlit.

## Funzionalità
- Upload CSV
- Anteprima dati
- Filtri dinamici (numerici e testuali)
- Analisi statistiche (media, somma, min, max…)
- Grafici (barre, linee, istogramma)

## Avvio applicazione
```bash
pip install -r requirements.txt
streamlit run app.py
